코드 규칙

주석

- 기본 주석 표시

- 시작과 끝의 표시는 _ / // 으로 표시한다.

- 수정 / 삭제 / 추가 는 날짜와 작성자, 목적을 함께 표시한다. 단, 한줄 주석시 닫힘 주석은 사용하지 않는다.
